# TASKANA Workplace

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 11.1.1.

## Install Dependencies

Run `yarn`. npm will not work

## Development server

Run `yarn start` for a dev server. Navigate to `http://localhost:1337/` to see the standalone mode. Please note, that a running backend of taskana in developement mode is required. For more information about that visit the [TASKANA wiki](https://taskana.atlassian.net/l/c/ZPnkT6i5)
